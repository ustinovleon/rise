# Rise Villas Phuket

Маркетинговый сайт проекта Rise Villas Phuket - премиальные виллы на Пхукете.

## Структура проекта

```
rise/
├── index.html          # Главная страница
├── legal/              # Правовые документы
│   ├── privacy.html           # Политика конфиденциальности
│   ├── personal-data-policy.html  # Политика обработки персональных данных (152-ФЗ)
│   ├── cookies.html           # Политика использования cookies
│   ├── terms.html             # Пользовательское соглашение
│   ├── consent.html           # Согласие на обработку персональных данных
│   └── requisites.html        # Реквизиты компании
├── scripts/            # Вспомогательные скрипты
└── CNAME               # Настройка домена
```

## Юридическая информация

Сайт содержит полный комплект правовых документов в соответствии с требованиями законодательства РФ:

- **Политика конфиденциальности** - порядок обработки и защиты информации пользователей
- **Политика обработки персональных данных** - соответствие 152-ФЗ "О персональных данных"
- **Политика использования cookies** - информация об использовании файлов cookie и Яндекс.Метрики
- **Пользовательское соглашение** - условия использования сайта
- **Согласие на обработку персональных данных** - текст согласия для пользователей
- **Реквизиты компании** - контактные и банковские реквизиты

## Аналитика

Сайт использует Яндекс.Метрику (ID: 106380735) с функциями:
- Вебвизор
- Карта кликов
- Отслеживание ссылок
- Точный показатель отказов

Информация об аналитике раскрыта в Политике использования cookies.
